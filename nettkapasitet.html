Metodikk for prioritetsscore

Hver bygning i Glitre Netts konsesjonsområde (Agder, Buskerud, deler av Østfold/Akershus, Hadeland, Askøy) får en prioritetsscore fra 0 til 100:

  1. Effektiviseringspotensial (50 %): samme beregning som på Energipotensial-kartet — registrert energiattest der den finnes, ellers estimat fra byggeår og type.
  2. Nettkapasitet (50 %): bygget kobles til nettkapasitet-sonen (trafoforsyningsområdet) det ligger i. Lav ledig kapasitet gir høy score på denne komponenten.

Prioritet = 0,5 × potensial + 0,5 × (100 − normalisert ledig kapasitet)

Nettkapasitet-dataene i dette kartet er syntetiske demo-data — DataArena (Glitre Netts ekte kapasitetskart, kart.dataarena.no) har ingen dokumentert åpen API for bulk-uttrekk. Se README for hvordan du bytter inn ekte tall manuelt per område.
